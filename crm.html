📋 Configuración en 5 pasos (una sola vez)

  1. Abre Google Sheets y crea un archivo llamado "prospectos"
  2. Ve a Extensiones → Apps Script
  3. Borra el código que aparece y pega el contenido del archivo apf_crm_script.gs que descargaste
  4. Click en Implementar → Nueva implementación → Tipo: Aplicación web → Ejecutar como: Yo → Acceso: Cualquier persona → Implementar
  5. Copia la URL de la aplicación web y pégala arriba
